The Guardian Journey es un juego donde podrás escoger o crear tu personaje para adentrarte hasta esta
aventura en la cual podrás recorrer ciudades y enfrentarte a guardianes para al final, cuando llegues a la 
capital Tesla y cumplas con los requisitos (Haber recorrido todas las villas y tener 90 puntos o más) podrás 
ser digno de enfrentarte al jefe Stormheart, aunque esta pelea la dejaremos para otra oportunidad.

Este programa fue escrito en visual studio code en el lenguaje de C++ ocupando distintas estructuras como 
mapas, gráficos, vectores, punteros entre otros. Las dos estructuras principales que componen al juego son 
los guardianes (un grafo) y las villas (un grafo creado mediante un mapa desordenado), ambas estructuras 
leídas mediante un .txt encontrado en este mismo repositorio.

Para poder jugar este gran juego deberá abrir el ejecutable “TheGuardianJourney.exe” el cual deberá poder
abrirse en un sistema operativo windows, si esta opción no es compatible con su dispositivo pruebe creando 
un nuevo ejecutable mediante el terminal de visual studio code ocupando el comando "g++ .\Proyecto2_Algoritmos.cpp"
y tendiendo instalado MinGW-w64 en sus sitema.
